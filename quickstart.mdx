## Prerequisites

You need **Node.js 18.x or later** installed before you begin. Node.js 20.x or later is recommended.

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/TailAdmin/free-nextjs-admin-dashboard.git
    ```

    <Note>
      **Windows users:** if you encounter issues while cloning, place the repository near the root of your drive (for example `C:\projects\`) to avoid path length limits.
    </Note>
  </Step>

  <Step title="Install dependencies">
    <CodeGroup>
      ```bash npm
      npm install
      ```

      ```bash yarn
      yarn install
      ```

      ```bash pnpm
      pnpm install
      ```
    </CodeGroup>

    <Note>
      If you see a peer-dependency error during installation, add the `--legacy-peer-deps` flag:

      ```bash
      npm install --legacy-peer-deps
      ```
    </Note>
  </Step>

  <Step title="Start the development server">
    ```bash
    npm run dev
    ```

    Next.js starts a local development server with hot-module replacement enabled.
  </Step>

  <Step title="Open the app in your browser">
    Navigate to [http://localhost:3000](http://localhost:3000). You should see the TailAdmin ecommerce dashboard.
  </Step>
</Steps>

## Available scripts

All scripts are defined in `package.json` and run via your package manager.

| Script | Command | Description |
|---|---|---|
| Development | `npm run dev` | Starts the Next.js dev server at `localhost:3000` |
| Build | `npm run build` | Compiles the app for production |
| Start | `npm run start` | Serves the production build locally |
| Lint | `npm run lint` | Runs ESLint across the entire project |
