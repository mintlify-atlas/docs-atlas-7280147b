    - Upgraded to Next.js 16.x to address [CVE-2025-29927](https://nextjs.org/blog/cve-2025-29927).
    - Added package overrides for `@react-jvectormap` to prevent peer-dependency errors during installation.
    - Migrated from `react-flatpickr` to the `flatpickr` package for React 19 support.
  </Accordion>

  <Accordion title="Version 2.0.1 — February 27, 2025">
    **Tailwind CSS v4 upgrade**

    - Upgraded to Tailwind CSS v4 for better performance and efficiency.
    - Updated class usage to match the latest v4 syntax and features.
    - Replaced deprecated classes and optimized styles.

    After pulling this update, run `npm install` (or `yarn install`) to update dependencies. Refer to the [Tailwind CSS v4 Migration Guide](https://tailwindcss.com/docs/upgrade-guide) if you encounter style changes or compatibility issues.
  </Accordion>

  <Accordion title="Version 2.0.0 — February 2025">
    **Major release — Next.js 16 redesign**

    This release is a complete redesign focused on Next.js 16 and React 19.

    **Major improvements**
    - Complete redesign using Next.js 16 App Router and React Server Components.
    - Enhanced user interface with Next.js-optimized components.
    - Improved responsiveness and accessibility.
    - New collapsible sidebar, chat screens, and calendar.
    - Redesigned authentication using Next.js App Router and Server Actions.
    - Updated data visualization using ApexCharts for React.

    **Breaking changes**
    - Migrated from Next.js 14 to Next.js 16.
    - Chart components now use ApexCharts for React (previously a different library).
    - Authentication flow updated to use Server Actions and middleware.

    See the [upgrade guide](/guides/upgrade-guide) for migration instructions.
  </Accordion>
