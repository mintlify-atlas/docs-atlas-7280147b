## What is TailAdmin?

TailAdmin is a production-ready admin dashboard template that gives you everything you need to build a back-end panel or admin interface without starting from scratch. It combines **Next.js 16** App Router, **React 19**, **TypeScript**, and **Tailwind CSS v4** into a cohesive, well-structured starting point.

The free version ships with one fully featured ecommerce dashboard, 30+ dashboard components, 50+ UI elements, authentication pages, and all the building blocks you need to create a professional admin panel.

## Key features

<CardGroup cols={3}>
  <Card title="Ecommerce Dashboard" icon="chart-line">
    Metrics, monthly sales chart, target progress ring, statistics chart, recent orders table, and demographic world map — all pre-wired.
  </Card>
  <Card title="ApexCharts Integration" icon="chart-bar">
    Responsive line and bar charts powered by ApexCharts for React, with light and dark mode support.
  </Card>
  <Card title="Collapsible Sidebar" icon="sidebar">
    Icon-only collapse, hover-to-expand, and mobile drawer — all handled by `SidebarContext`.
  </Card>
  <Card title="Dark Mode" icon="moon">
    System-aware dark mode toggle using `ThemeContext` and Tailwind's `dark:` variant classes.
  </Card>
  <Card title="Auth Pages" icon="lock">
    Sign In and Sign Up pages built with Next.js Server Actions and App Router layout nesting.
  </Card>
  <Card title="Form Components" icon="input-text">
    Input, Select, MultiSelect, DatePicker (Flatpickr), Switch, and grouped inputs — all styled with Tailwind.
  </Card>
  <Card title="UI Elements" icon="shapes">
    Alerts, Avatars, Badges, Buttons, Images, Videos, Modals, and Dropdowns — ready to drop in.
  </Card>
  <Card title="FullCalendar" icon="calendar">
    Interactive calendar page with day, week, list, and time-grid views via `@fullcalendar/react`.
  </Card>
  <Card title="TypeScript Throughout" icon="code">
    Fully typed components and contexts — no `any` shortcuts, real interfaces everywhere.
  </Card>

| Technology | Version | Role |
|---|---|---|
| Next.js | 16.x | App Router, SSR, Server Actions |
| React | 19.x | Component model |
| TypeScript | 5.x | Type safety |
| Tailwind CSS | v4 | Utility-first styling |
| ApexCharts | 4.x | Data visualization |
| FullCalendar | 6.x | Calendar component |
| Flatpickr | 4.x | Date picking |
| jVectorMap | — | World map component |
