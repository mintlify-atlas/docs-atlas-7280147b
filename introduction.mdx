title: "Introduction"
description: "TailAdmin is a free, open-source admin dashboard template built with Next.js 16, React 19, TypeScript, and Tailwind CSS v4."
---

TailAdmin gives you a complete starting point for building admin dashboards and back-end interfaces. It ships with production-ready UI components, page layouts, and data visualization tools so you can focus on your application logic rather than rebuilding common UI patterns from scratch.

<CardGroup cols={2}>
  <Card title="Dashboard components" icon="chart-bar">

  </Card>

    50+ UI elements: alerts, avatars, badges, buttons, dropdowns, modals, and more — all styled with Tailwind CSS.
  </Card>
  <Card title="Authentication pages" icon="lock">
    Sign in and sign up pages built with Next.js App Router and server actions, ready to connect to your auth backend.

    Responsive sidebar with expand/collapse, hover-expand, and mobile drawer behaviour managed by `SidebarContext`.
  </Card>

## Technology stack

| Technology | Version | Role |
|---|---|---|
| [Next.js](https://nextjs.org) | 16.x | Framework — App Router, SSR, SSG, API routes |
| [React](https://react.dev) | 19.x | UI rendering and component model |
| [TypeScript](https://www.typescriptlang.org) | 5.x | Static typing across the entire codebase |
| [Tailwind CSS](https://tailwindcss.com) | v4 | Utility-first styling with custom design tokens |
| [ApexCharts](https://apexcharts.com) | 4.x | Interactive line and bar charts |
| [FullCalendar](https://fullcalendar.io) | 6.x | Full-featured calendar with day, week, and list views |
| [Flatpickr](https://flatpickr.js.org) | 4.x | Lightweight date and time picker |
| [jVectorMap](https://github.com/bjornd/jvectormap) | — | Interactive world map component |

## Free vs Pro

The free version covered by this documentation includes one dashboard, 30+ components, and 50+ UI elements. A Pro version is available with seven unique dashboards, 500+ components, a complete Figma design file, and email support.
