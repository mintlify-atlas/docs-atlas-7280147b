The ecommerce dashboard is the root page of TailAdmin (`app/(admin)/page.tsx`). It composes six self-contained widgets into a responsive `grid-cols-12` layout using Tailwind CSS. Each widget lives under `src/components/ecommerce/` and is a standalone React component you can use independently.

## Page layout

```tsx
// src/app/(admin)/page.tsx
import type { Metadata } from "next";
import { EcommerceMetrics } from "@/components/ecommerce/EcommerceMetrics";
import React from "react";
import MonthlyTarget from "@/components/ecommerce/MonthlyTarget";
import MonthlySalesChart from "@/components/ecommerce/MonthlySalesChart";
import StatisticsChart from "@/components/ecommerce/StatisticsChart";
import RecentOrders from "@/components/ecommerce/RecentOrders";
import DemographicCard from "@/components/ecommerce/DemographicCard";

export const metadata: Metadata = {
  title: "Next.js E-commerce Dashboard | TailAdmin - Next.js Dashboard Template",
  description: "This is Next.js Home for TailAdmin Dashboard Template",
};

export default function Ecommerce() {
  return (
    <div className="grid grid-cols-12 gap-4 md:gap-6">
      <div className="col-span-12 space-y-6 xl:col-span-7">
        <EcommerceMetrics />
        <MonthlySalesChart />
      </div>

      <div className="col-span-12 xl:col-span-5">
        <MonthlyTarget />
      </div>

      <div className="col-span-12">
        <StatisticsChart />
      </div>

      <div className="col-span-12 xl:col-span-5">
        <DemographicCard />
      </div>

      <div className="col-span-12 xl:col-span-7">
        <RecentOrders />
      </div>
    </div>
  );
}
```

### Grid structure summary

| Column span (xl) | Widget |
|---|---|
| 7 | `EcommerceMetrics` + `MonthlySalesChart` (stacked) |
| 5 | `MonthlyTarget` |
| 12 | `StatisticsChart` |
| 5 | `DemographicCard` |
| 7 | `RecentOrders` |

## EcommerceMetrics

Displays two KPI cards side-by-side — **Customers** (3,782, +11.01%) and **Orders** (5,359, -9.05%). Each card shows an icon, a label, a numeric value, and a colour-coded `Badge` indicating the period-over-period trend.

```tsx
import { EcommerceMetrics } from "@/components/ecommerce/EcommerceMetrics";

<EcommerceMetrics />
```

<Note>
`EcommerceMetrics` is a `"use client"` component. It contains no props — replace the hardcoded values with your own data fetching logic when building for production.
</Note>

## MonthlySalesChart

A bar chart rendered with **ApexCharts** showing monthly sales figures across all 12 months of the year. Includes a contextual `Dropdown` menu (View More / Delete) in the top-right corner. The chart is dynamically imported with `ssr: false` to avoid hydration issues.

```tsx
import MonthlySalesChart from "@/components/ecommerce/MonthlySalesChart";

<MonthlySalesChart />
```

**Key implementation details:**

- Uses `dynamic(() => import("react-apexcharts"), { ssr: false })` for the chart.
- Bar chart type with `columnWidth: "39%"`, `borderRadius: 5`, and brand colour `#465fff`.
- Month categories on the x-axis; sales data array drives the bars.
- Internal `isOpen` state controls the dropdown visibility.

## MonthlyTarget

A radial (semi-circle) progress chart showing the current month's revenue vs. target. Displays a percentage in the centre of the gauge and a summary row of **Target**, **Revenue**, and **Today** figures below. Includes a `Dropdown` menu.

```tsx
import MonthlyTarget from "@/components/ecommerce/MonthlyTarget";

<MonthlyTarget />
```

**Key implementation details:**

- Radial bar chart (`type: "radialBar"`) via ApexCharts, dynamically imported.
- `series = [75.55]` represents 75.55% completion.
- `startAngle: -85`, `endAngle: 85` creates the half-circle shape.
- A `+10%` badge is absolutely positioned inside the chart area.

---

## StatisticsChart

A dual-series area/line chart comparing **Sales** and **Revenue** trends over 12 months. Includes a `ChartTab` filter component and a `flatpickr` date range picker in the header.

```tsx
import StatisticsChart from "@/components/ecommerce/StatisticsChart";

<StatisticsChart />
```

**Key implementation details:**

- Chart type is `"area"` with `stroke.curve: "straight"`.
- Two series: `Sales` (blue `#465FFF`) and `Revenue` (light blue `#9CB9FF`).
- `flatpickr` date range picker is initialised in a `useEffect` and defaults to the last 7 days.
- The chart wrapper uses `min-w-[1000px] xl:min-w-full` so it scrolls horizontally on narrow viewports.

## RecentOrders

A data table listing the 5 most recent orders. Columns: **Product** (image + name + variant count), **Category**, **Price**, and **Status**. Status is rendered as a colour-coded `Badge` (`success`, `warning`, or `error`).

```tsx
import RecentOrders from "@/components/ecommerce/RecentOrders";

<RecentOrders />
```

**Internal data shape:**

```typescript
interface Product {
  id: number;
  name: string;
  variants: string;       // e.g. "2 Variants"
  category: string;
  price: string;          // e.g. "$2399.00"
  image: string;          // path to product image
  status: "Delivered" | "Pending" | "Canceled";
}
```

The component uses TailAdmin's `Table`, `TableHeader`, `TableBody`, `TableRow`, and `TableCell` primitives from `@/components/ui/table`.

## DemographicCard

Displays customer distribution by country. Features an interactive SVG world map (`CountryMap`) and a list of countries with a progress bar and percentage figure. Includes a `Dropdown` menu.

```tsx
import DemographicCard from "@/components/ecommerce/DemographicCard";

<DemographicCard />
```

**Key implementation details:**

- `CountryMap` is a separate SVG-based map component imported from `./CountryMap`.
- Country flag images are sourced from `/images/country/country-0x.svg`.
- Progress bars are pure Tailwind CSS (no third-party library).
- Internal `isOpen` state manages the `Dropdown`.
